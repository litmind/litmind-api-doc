# LightboxItem

An item stored inside a lightbox.

```json
{
    "entity": "LightboxItem",
    "id": 761540,
    "note": null,
    "added": "2025-02-21T13:21:12+00:00",
    "item": <A Profile or Artwork entity>
},
```
